ohmage-omh is an open-source, open-architecture, mobile health platform intended for rapid prototyping and piloting of mobile health techniques. The system consists of downloadable applications for both Android and iOS, and a secure Open mHealth compliant data store. It supports mobile phone-based data capture through both passive data collection and simple surveys. Data are securely stored for analysis and visualization using Open mHealth based APIs.

To date there are three mobile data collection applications supported on both iOS and Android:

  • The ohmage app is currently available for Android (3.0.x+) and iOS (XX+) devices. It supports simple surveys with reminders. Researchers author surveys in JSON or send text based surveys to our team for conversion.
  • Mobility provides continuous passive measurement of activity and location (iOS 5S+).
  • PAM is a visual tool for affect/mood self-report
